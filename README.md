🔹 1. تفاوت اجرای حلقه‌ها در سطح Interpreter و Bytecode پایتون

Interpreter (مفسر) همون برنامه‌ایه که کد پایتون تو رو می‌خونه و اجرا می‌کنه.

وقتی یه کد می‌نویسی، پایتون اول اون رو به Bytecode (کد میانی) تبدیل می‌کنه.

بعد موتور CPython اون bytecode رو اجرا می‌کنه.


🔸 یعنی کد پایتون مستقیم اجرا نمی‌شه، بلکه به دستورات سطح پایین‌تری تبدیل می‌شه.
🔸 در این سطح، حلقه‌ها (for, while) تبدیل به دستوراتی می‌شن که پرش (jump) بین خطوط انجام می‌دن.

🔹 3. تفاوت Comprehensionها با حلقه سنتی

Comprehensionها مثل حلقه هستن ولی در یک خط نوشته می‌شن و بهینه‌تر اجرا می‌شن:

# روش معمولی
squares = []
for i in range(10):
    squares.append(i**2)

# روش comprehension
squares = [i**2 for i in range(10)]

✅ مزیت‌ها:

سریع‌تر هستن (به خاطر اجرای مستقیم در C level)

حافظه کمتری مصرف می‌کنن

کد خواناتر می‌شه

🔹 5. مفهوم Iterator و Iterable

Iterable: هر چیزی که می‌تونی روش تکرار انجام بدی (مثل list، tuple، string)

Iterator: ابزاری که از داخل iterable مقدارها رو یکی‌یکی بیرون می‌کشه.


nums = [1, 2, 3]
it = iter(nums)  # ساخت iterator
print(next(it))  # 1
print(next(it))  # 2

🔹 حلقه for در پایتون دقیقاً از همین iter() و next() برای تکرار استفاده می‌کنه.
